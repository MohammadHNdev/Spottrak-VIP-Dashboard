معرفی پروژه

این پروژه یک پنل مدیریت تحت وب قدرتمند و کاربرپسند است که به طور خاص برای مدیریت کاربران VIP ربات تلگرام اسپات‌ترک (Spottrak) طراحی و توسعه یافته بود. هدف اصلی این پنل، فراهم کردن ابزاری بصری و کارآمد برای مدیر جهت مشاهده، افزودن، تمدید و حذف کاربران VIP و همچنین نمایش آمارهای کلیدی مربوط به آن‌ها بوده است.

با وجود عملکرد فنی بی‌نقص، به دلیل محدودیت‌های اعمال شده از سوی پلتفرم اسپاتیفای و تمرکز تیم بر پروژه‌های جدید، فعالیت سرویس متوقف گردید و اکنون کد منبع آن جهت استفاده، مطالعه و یادگیری جامعه توسعه‌دهندگان منتشر می‌شود.

ویژگی‌های کلیدی

  • مدیریت جامع کاربران VIP: امکان افزودن، تمدید و حذف کاربران.
  • جدول تعاملی کاربران: نمایش لیست کاربران با قابلیت‌های جستجو، مرتب‌سازی، صفحه‌بندی و خروجی Excel/CSV.
  • مشاهده جزئیات کاربر: دسترسی به اطلاعات کامل هر کاربر شامل تاریخ شمسی، مدت اشتراک و تاریخچه تمدیدها.
  • نمایش آمارهای کلیدی: ارائه خلاصه‌ای از وضعیت کاربران (تعداد کل، فعال و منقضی شده).
  • نمودارهای بصری: نمایش توزیع وضعیت کاربران و مدت اشتراک‌ها با استفاده از Chart.js.
  • رابط کاربری مدرن و واکنش‌گرا: طراحی زیبا با Bootstrap 5.3 RTL و فونت وزیرمتن.
  • پایگاه داده قدرتمند: استفاده از پایگاه داده NoSQL MongoDB برای انعطاف‌پذیری بالا.

تکنولوژی‌های استفاده شده

بک‌اند (Backend)

  • Python & Flask
  • PyMongo (برای ارتباط با MongoDB)
  • jdatetime (برای کار با تاریخ شمسی)
  • Gunicorn (برای استقرار)

فرانت‌اند (Frontend)

  • HTML5, CSS3, JavaScript
  • jQuery & Bootstrap 5.3 RTL
  • DataTables (برای جدول تعاملی)
  • Chart.js (برای نمودارها)

راه‌اندازی و نصب

برای اجرای پروژه، مراحل زیر را دنبال کنید:

۱. کلون کردن مخزن

# Clone the repository from GitHub
git clone https://github.com/MohammadHNdev/Spottrak-VIP-Dashboard.git
cd Spottrak-VIP-Dashboard/vip_panel

۲. ایجاد و فعال‌سازی محیط مجازی

# Create a virtual environment
python -m venv venv
# On Windows:
venv\Scripts\activate
# On Linux/macOS:
source venv/bin/activate

۳. نصب وابستگی‌ها

pip install -r requirements.txt

۴. پیکربندی اتصال به MongoDB

فایل backend/app.py را باز کرده و رشته اتصال خود را در متغیر MONGO_URI جایگزین کنید.

MONGO_URI = "YOUR_MONGODB_CONNECTION_STRING"

۵. تنظیم کلید مخفی و اطلاعات ادمین

در همان فایل backend/app.py، متغیر app.secret_key را به یک رشته امن و تصادفی تغییر دهید. همچنین منطق ورود ادمین را برای محیط پروداکشن امن‌سازی کنید.

app.secret_key = 'یک_کلید_مخفی_بسیار_امن_اینجا_قرار_دهید'
# ...
if username == 'admin' and password == 'password':
    # هشدار: این بخش را با یک سیستم احراز هویت امن جایگزین کنید

۶. اجرای برنامه

# For development
python backend/app.py
# For production with Gunicorn
gunicorn -w 4 app:app -b 0.0.0.0:5000

نحوه استفاده

پس از راه‌اندازی موفقیت‌آمیز برنامه، به آدرس پنل در مرورگر خود بروید (مثلاً http://127.0.0.1:5000). ابتدا باید با استفاده از اطلاعات ورود ادمین که در مرحله پیکربندی تنظیم کرده‌اید، وارد شوید.

پس از ورود، به صفحه داشبورد هدایت می‌شوید که می‌توانید لیست کاربران VIP را مشاهده کرده و عملیات افزودن، تمدید و حذف را از طریق دکمه‌های مربوطه انجام دهید.
